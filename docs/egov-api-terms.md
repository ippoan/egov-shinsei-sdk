# e-Gov 電子申請API 利用規約について

本ライブラリは e-Gov 電子申請API を利用するクライアントソフトウェアです。

## 利用規約の確認

電子申請APIの利用にあたっては、デジタル庁が定める利用規約への同意が必要です。

- 規約全文: https://developer.e-gov.go.jp/contents/specification/shinsei-api/shinsei-api-terms.html
- 開発者ポータル: https://developer.e-gov.go.jp/

## 本ライブラリ利用者への注意

- APIキーの取得・最終確認試験の実施が必要です（利用規約 第3条, 第9条）
- APIキーを第三者に譲渡・開示してはなりません（第7条2項）
- 検証環境はテスト目的にのみ利用可能です（第12条1号）

## 知的財産について（第15条）

- 本ライブラリ（API利用ソフトウェア）の著作権は開発者に帰属します（第15条1項）
- 電子申請API仕様・ドキュメントの知的財産権はデジタル庁に帰属します（第15条2項）
- 本ライブラリにはAPI仕様書・openapi.jsonを同梱していません
